Hello MCP (Model Context Protocol)

Hello-MCP는 Claude Desktop에서 Model Context Protocol(MCP)을 통해 외부 도구와 상호작용할 수 있도록 도와주는 유틸리티 도구입니다. 셋업부터 MCP 서버 운영, 샘플 도구 활용까지 한 번에 제공하여 Claude AI와 사용자 정의 도구 간의 원활한 통합을 지원합니다.

왜 Hello-MCP가 필요한가요?

LLM(인공지능)을 외부 세계(이메일, 데스크탑 유틸리티 앱)와 연결함으로써, 단순히 대화만 하던 것에서 벗어나 여러분이 이미 사용하던 다양한 도구들과 쉽게 연결할 수 있는 방법을 알게 되는 계기가 되길 바랍니다.

🛠️ 주요 기능​

  • 간편한 설정: 단일 명령어로 Claude Desktop MCP 통합 설정 완료
  • 다양한 실행 모드: 셋업 마법사, MCP 서버, CLI 도구 등 다양한 모드로 실행 가능
  • 내장 샘플 도구: 이메일 전송, 간단한 계산 등 사전 구성된 도구 제공
  • 확장성: 사용자 정의 MCP 도구 쉽게 추가 가능
  • 다국어 지원: 한국어와 영어 지원

🎬 데모 영상​

Hello-MCP의 설정 및 사용 방법을 시연하는 데모 영상입니다. 아래 이미지를 클릭하면 영상이 재생됩니다.

📋 요구 사항​

  • Node.js 14+ 다운로드
  • Claude Desktop 설치 (다운로드)
  • 이메일 전송 기능을 위한 인터넷 연결 및 GARAK API 키 생성

🚀 빠른 시작​

설치 없이 npx로 Hello-MCP를 실행할 수 있습니다:

npx hello-mcp

이 명령어는 전체 구성 프로세스를 안내하는 설정 마법사를 시작합니다.

🔄 실행 흐름​

  1. 명령어 실행:

    npx hello-mcp

  2. 진행 단계:

    • Claude Desktop 설치 여부 확인
    • 이메일 입력 → API Key 생성
    • MCP 환경 자동 설정
    • Claude Desktop 자동 재시작 (가능한 경우)
    • 셋업 완료

  3. 셋업 완료 후 가능한 경험:

    • Claude에 계산 요청: "1 add 1"
    • Claude에게 이메일 보내기: "Send an email to me with the result of 1 add 1"
    • MCP 서버 모드로 전환: npx hello-mcp mcp-server
    • 커스텀 툴 추가 가능

🔰 Windows 비개발자를 위한 시작 가이드​

처음 사용하는 비개발자를 위한 단계별 설정 방법입니다:

  1. Claude Desktop 설치

    • Claude.ai 다운로드 페이지에서 Claude Desktop을 다운로드하고 설치합니다.
    • 설치 후 Claude Desktop을 실행하고 계정에 로그인합니다.

  2. Node.js 설치

    • Node.js 공식 웹사이트에서 LTS 버전을 다운로드합니다.
    • 다운로드한 설치 파일을 실행하고 안내에 따라 설치를 완료합니다.
    • 기본 설정으로 진행하면 됩니다 (특별한 설정 불필요).

  3. Hello-MCP 실행

    • 윈도우 검색에서 'cmd'를 입력하여 명령 프롬프트를 실행합니다.
    • 명령 프롬프트 창에서 다음 명령어를 입력합니다: 
      npx -y hello-mcp
    • 화면에 표시되는 안내에 따라 진행합니다.

완료 후 Claude Desktop에서 계산 요청이나 이메일 전송과 같은 기능을 사용해 볼 수 있습니다!
